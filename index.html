Ciencia de Datos

Por Marco Antonio Hidalgo Encina

¿Qué es ciencia de datos?

Es un enfoque multidisciplinario que combina principios y prácticas del campo de las matemáticas, la estadística, la inteligencia artificial y la ingeniería de computación para analizar grandes cantidades de datos.

¿Por qué utilizar Python en ciencia de datos?

Porque permite extraer información valiosa de bases de datos y analizar datos de forma rápida y eficiente. Python cuenta con una gran cantidad de bibliotecas, o librerías, que permiten procesar y visualizar datos.

Pandas

¿Qué es Pandas?

La biblioteca de software de código abierto Pandas está diseñada específicamente para la manipulación y el análisis de datos en el lenguaje Python. Es potente, flexible y fácil de usar.

¿Cómo funciona?

Pandas trabaja sobre «DataFrames«: tablas de datos bidimensionales, donde cada columna contiene los valores de una variable y cada fila contiene un conjunto de valores de cada columna. Los datos almacenados en un DataFrame pueden ser números o caracteres.

Ventajas

Para los Data Scientists y desarrolladores, Pandas aporta varias ventajas. Esta biblioteca permite compensar fácilmente los datos que faltan. Es una herramienta flexible, ya que las columnas pueden insertarse o eliminarse fácilmente dentro de los DataFrames. La alineación de los datos con las etiquetas puede automatizarse.

Cómo utilizan Pandas los científicos de datos

Los Data Scientists y los programadores familiarizados con el lenguaje de programación R para cálculo estadístico utilizan DataFrames para almacenar datos en una cuadrícula muy sencilla de revisar. Por eso Pandas se utiliza mucho para Machine Learning.
